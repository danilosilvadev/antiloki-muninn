  1. Open ⚙ Settings. Paste SUPABASE_DB_URL (dashboard → Connect → Session pooler URI) → save + reload → the db dot goes green, Leads/Dashboard light up.
  2. Paste OPENROUTER_API_KEY and FULLENRICH_API_KEY → save → pipeline workers go green: the raven can fly.
  3. Optional now: APOLLO_API_KEY (powers “find similar”), CALENDLY_URL (the drawer's book-a-call button).
  4. Telegram (still the phone console): paste TELEGRAM_BOT_TOKEN, save, message the bot once, paste the chat id it replies with into TELEGRAM_OPERATOR_CHAT_ID, save again.
where the keys actually live (G4, kept honest)
