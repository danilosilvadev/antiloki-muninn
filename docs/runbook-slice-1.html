05Boundaries & the flag log

  • Nothing sends automatically. ✅ queue only stages a lead; Smartlead + SendPolicy + the review queue are slice 3. The bot messages you, nobody else.
  • Costs at this slice: ~$0.06/enriched lead (FullEnrich) + ~$0.01–0.04/dossier (OpenRouter mid-tier) → 300 leads ≈ $30–40 total.
  • Data honesty: the analysis prompt is hard-scoped to role-relevant professional signals, evidence quoted verbatim from the enrichment or absent, "thin data" said out loud. If a dossier ever reads as more than that, that's a bug — file it.
